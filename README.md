# Lernlandschaft Iran

Eigenständige interaktive Lernlandschaft zu Iran mit vier Stationen:

1. Perserreich, Monarchie und geopolitische Lage
2. Mossadegh, Ölkonflikt und Staatsstreich 1953
3. Mohammad Reza Pahlavi, Modernisierung und Revolution
4. Chomeini, Islamische Republik und Gegenwart

## Inhaltlicher Fokus

- historische Tiefenlinien statt reiner Gegenwartsfixierung
- Konflikte, Kultur und geopolitische Bedeutung zusammendenken
- neue Situationen mit einem Update-Kompass einordnen

## Start

Die Lernlandschaft ist eine statische Webanwendung. `index.html` im Browser öffnen.

## Struktur

- `index.html`: Einstieg und Seitenstruktur
- `app.js`: Interaktionslogik, Sofortkorrektur, Lehrer*innenmodus
- `data.js`: Module, Quellen und Fragen
- `styles.css`: Layout und Gestaltung
